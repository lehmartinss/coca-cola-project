1- O que é um repositório no Git e qual a sua importância?
Um repositório no Git é um local onde o código-fonte e o histórico de desenvolvimento de um projeto são armazenados.

2- Qual a função do Git Bash?
O Git Bash é uma interface de linha de comando que permite aos usuários executarem comandos do Git.

3- O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
O comando git add adiciona mudanças no repositório local para o próximo commit. git add <arquivo> adiciona um arquivo 
específico, enquanto git add . adiciona todos os arquivos modificados na pasta atual.

4- Por que utilizamos o ponto (.) no comando git add .?
O ponto (.) no comando git add . indica que todos os arquivos na pasta atual devem ser adicionados.

5- Explique o comando git commit e por que é importante criar uma mensagem descritiva.
O git commit salva as mudanças adicionadas no histórico do repositório, é importante para entender o que foi alterado 
em cada commit.

6- Qual a função do comando git push?
O git push envia os commits locais para um repositório remoto, permitindo que outras pessoas vejam as atualizações.

7- Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
Commits frequentes ajudam a manter o histórico do projeto detalhado.

8- Explique a função das tags HTML e como elas estruturam uma página.
As tags HTML são usadas para estruturar e organizar o conteúdo de uma página web, indicando cabeçalhos, parágrafos, links,
imagens e outros elementos.

9- O que são atributos em uma tag HTML e qual a sua função?
Atributos fornecem informações adicionais sobre os elementos HTML, como classes, identificadores, links

10- O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
Emmet é uma ferramenta que permite escrever código HTML e CSS mais rapidamente com atalhos.

11- O que é o box-sizing no CSS e por que usamos ele no reset inicial?
box-sizing define como o tamanho de elementos, no reset é utilizado para simplificar o cálculo de larguras e alturas.

12- Por que é importante utilizar tags semânticas no HTML?
Tags semânticas tornam o código mais compreensível.

13- O que são media queries no CSS e qual a sua função?
Media queries permitem aplicar estilos diferentes dependendo das características do dispositivo.

14- Qual a função do GitHub Pages?
O GitHub Pages permite hospedar sites estáticos diretamente de um repositório GitHub.

15- O que é o Flexbox no CSS e para que ele é utilizado?
Flexbox é um layout CSS que facilita o alinhamento e distribuição de espaço entre elementos em um container.

16- Qual a diferença entre justify-content e align-items em um container Flexbox?
justify-content alinha os itens no eixo principal (horizontal por padrão), enquanto align-items alinha no eixo perpendicular 
(vertical por padrão).

17- Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
flex-grow controla o quanto um item pode crescer em relação aos outros no eixo principal, útil para distribuir espaço extra.

18- Explique a diferença entre flex-direction: column; e flex-direction: row;.
flex-direction: column; organiza os itens verticalmente, enquanto flex-direction: row; organiza-os horizontalmente.
